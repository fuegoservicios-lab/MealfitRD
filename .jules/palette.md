## 2024-07-02 - Redundant Screen Reader Announcements in Icon Buttons\n**Learning:** When adding `aria-label` to icon-only buttons that use inline SVGs (like Lucide React components), screen readers may still announce the internal SVG elements redundantly if they aren't explicitly hidden.\n**Action:** Always add `aria-hidden="true"` to the internal SVG component (e.g., `<Check size={15} aria-hidden="true" />`) when adding an `aria-label` to its parent button.
