## 2024-05-18 - [Hoist Static Data and Precompile RegExp in React Components]
**Learning:** Found static arrays and sets defined inside a large React component (`Dashboard.jsx`), notably the `stops` array inside a normalization function and `PANTRY_STAPLES_DELTA`. More importantly, `new RegExp` was being instantiated inside a `for` loop dynamically on every call to the normalization function during rendering. This causes excessive object instantiation, garbage collection pressure, and CPU overhead.
**Action:** Extract static arrays, sets, and compile `RegExp` objects outside of the React component body to prevent re-creation on every render. Combine multiple exact string matches into a single precompiled `RegExp` (e.g. `join('|')`) to replace loop-based dynamic regex instantiation. This turns an O(N) regex evaluation process into O(1) and eliminates runtime compilation overhead.
